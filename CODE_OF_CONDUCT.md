# Code of Conduct

## Our Pledge

We want BookForge to be a welcoming project for people building local-first audiobook tooling.

Contributors are expected to be respectful, constructive, and patient.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing private documents, audio, credentials, or paths from another user.
- Spam or unrelated promotion.
- Repeatedly ignoring maintainer guidance.

## Enforcement

Maintainers may edit, close, or remove issues, pull requests, and comments that violate this code of conduct.
